GitExtensions: Solution Runner

  • Source code at https://github.com/maraf/GitExtensions.SolutionRunner .
  • Release of the GitExtensions: Solution Runner v4.2.1 to download.
  • Current version on NuGet at nuget.org/packages/GitExtensions.SolutionRunner.

Description

This extension for GitExtensions adds a list of Visual Studio solution files and VS code workspaces contained in the repository.
