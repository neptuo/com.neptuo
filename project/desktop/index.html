Our projects!

Apps, libs & tools that helps during every day (developer) life.

Chromium Updater

  • Chromium updater is Click-once application for downloading and updating latest builds of Chromium. The application offers automatic or manual download of latest build to selected folder and then extracting it there.
  • Source code at https://github.com/maraf/ChromiumUpdater.
  • Release of the Chromium Updater v1.0.2 to download.

GitExtensions: Bundle Backuper

  • Branch backuping plugin for GitExtensions.
  • Source code at https://github.com/maraf/GitExtensions.BundleBackuper.
  • Release of the GitExtensions: Bundle Backuper v1.0.0 to download.
  • Current version on NuGet at nuget.org/packages/GitExtensions.BundleBackuper.

GitExtensions: Solution Runner

  • A Visual Studio solution runner plugin for Git Extensions.
  • Source code at https://github.com/maraf/GitExtensions.SolutionRunner.
  • Release of the GitExtensions: Solution Runner v5.0.0 to download.
  • Current version on NuGet at nuget.org/packages/GitExtensions.SolutionRunner.

Link Extractor

  • Desktop application for extracting links (URLs) from HTML or any other document type.
  • Source code at https://github.com/maraf/LinkExtractor.
  • Release of the Link Extractor v1.0.1 to download.

Open Special Folder

  • Desktop application for browsing Windows special folders.
  • Source code at https://github.com/maraf/OpenSpecialFolder.
  • Release of the Open Special Folder v1.0.0 to download.

Plugin Manager

  • NuGet based univarsal plugin manager.
  • Source code at https://github.com/maraf/PluginManager.
  • Release of the Plugin Manager to download.

Solution Runner

  • Keyboard-only designed application for simple and fast opening of Visual Studio solution files.
  • Source code at https://github.com/neptuo/Productivity.SolutionRunner.
  • Release of the Solution Runner v2.0.0 to download.

Windows Dock

  • WindowsDock is OSX-like application launcher. It has many layout and keyboard configurations to suit most users needs. Beside application shortcuts, dock can host user notes with alarms, simple directory and desktop browser and custom command line scripts.
  • Source code at https://github.com/maraf/WindowsDock.
  • Release of the Windows Dock v1.2.22 to download.

WinRun

  • Simple utility application with (missing) Windows tweaks.
  • Source code at https://github.com/neptuo/Productivity.DesktopTools.
  • Release of the WinRun v1.10.1 to download.
