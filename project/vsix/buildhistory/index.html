Neptuo.Productivity.BuildHistory

  • Licensed under Apache License, Version 2.0.
  • Project documentation.
  • Source code at https://github.com/neptuo/Productivity.
  • Current version can be installed from Official VSIX gallery.

Description

Build history measures a length of every build (rebuild and clean). In the overview window it shows a configured number of last builds, also in the title there is a total count and total time ellapsed during builds. It gives you an overview how much time you spend during builds.

Version history

  • v1.1
    • Visual Studio 2019 support.
    • Blue bar during build.
    • Fixed default count of shown builds, no need to set it.
    • Bar length is now computed from lengths of previous builds.
  • v1.0
    • Configurable count of builds to store.
    • Each build length, result (success or failure) and type (build, rebuild or clean).
