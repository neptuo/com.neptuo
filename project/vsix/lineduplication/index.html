Neptuo.Productivity.LineDuplication

  • Selected line(s) duplication feature for VisualStudio.
  • Licensed under Apache License, Version 2.0.
  • Project documentation.
  • Source code at https://github.com/neptuo/Productivity .
  • Current version can be installed from Official VSIX gallery.

Version history

  • v1.1
    • Visual Studio 2019 support.
    • Edit menu commands has been moved under 'Advanced' sub-menu.
    • Fixed duplication of language comments (for C# '//' or '///').
  • v1.0
    • Single line duplication.
    • Multi line duplication.
    • Preserves selection in duplicated line(s).
    • Default key bindings: Ctrl+Shift+Alt+Up & Ctrl+Shift+Alt+Down.
