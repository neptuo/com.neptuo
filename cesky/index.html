Co vám můžeme nabídnout?

Potřebujete Windows aplikaci nebo dáváte přednost webovým aplikacím s bohatým uživatelským rozhraním, nebo snad okamžitá dostupnost mobilních aplikací je to, co vám vyhovuje? Můžeme vám pomoci ve všech těchto oblastech.

Podnikové aplikace

Aplikace postavené přímo na míru zákazníkovi, využívající naše již hotové komponenty. Cílem je zde optimalizace každodenních úkolů zákazníka.

Dodávané jako webové, mobilní nebo Windows aplikace.

Webové stránky

Prezentace firem nebo osob, postavené na našem redakčním systému, který toho již velmi mnoho umí. Můžeme hostovat několik webů v rámci jednoho redačkního systému a řídit přistup uživatelů k jednolivým částem.

Mobilní aplikace

Máme zkušenosti s aplikace pro Andriod i mobilní Windows. Můžeme dokonce použít Xamarin pro vývoj aplikace, která bude fungovat na všech platformách.

C#

Oblíbené

Nás oblíbenější programovací jazyk, podporující Windows a díky .NET Core také Linux a MAC OSX.

PHP

Levné

Náš redakční systému je postaven na PHP a díky levnému hostingu pro PHP, je ideální pro osobní nebo firemní webové presentace a malé podnikové aplikace.

K najmutí

Nebráníme se žádné spolupráci.

Můžeme nabídnout znalosti v C#, PHP, SQL a JavaScriptu s letitou praxí.

Open-source projecty

Většina naší infrastruktury je open-source, dostupná na Google Code a nyní na GitHubu. Tyto projekty prokrývají škálu od různých rozšíření základních knihoven až po plnou implementaci CQRS pro distribuované a škálovatelné aplikace.

Tyto projekty jsou nejčastěji dostupné jako NuGet balíčky .NET knihoven, rozšíření pro VisualStudio nebo ClickOnce Windows aplikace.

Poslední realizované webové prezentace

Zde je několik posledních realizovaných webových prezentací, využívající náš redakční systém.

Několik webů jsme též realizovali jako předgenerované statické stránky, hostované na bezplatných službách. Často jsou generovány z komplexních webových technologií, jako je ASP.NET, nicméně na server se publikují jen vygenerované HTML, obrázky a styly. Tyto weby pak mají vynimální nároky na server, na kterém běží.
